### 8. 비교 표 강화 (현황 및 문제점 섹션)

```
│   구분   │ 기존 혁신도시 │  제안 리빙랩 도시        │
│          │     (❌)     │        (✅)              │
├──────────┼──────────────┼──────────────────────────┤
│ 의사결정 │ 중앙 주도    │ 지역 주도 (상향식)       │
│          │ (하향식)     │                          │
├──────────┼──────────────┼──────────────────────────┤
│ 투자방식 │ LH 단독 부담 │ 민간 자본 활용           │
│          │ (부채 증가)  │ (레버리지 10배)          │
├──────────┼──────────────┼──────────────────────────┤
│ 접근방식 │ 완성형 개발  │ 리빙랩 실험→확산         │
│          │ (일괄 조성)  │ (단계적 검증)            │
├──────────┼──────────────┼──────────────────────────┤
│ 산업전략 │ 획일적 모델  │ 지역 특화 맞춤           │
│          │              │ (4개 유형)               │
├──────────┼──────────────┼──────────────────────────┤
│ 주민참여 │ 사후 이주    │ 기획부터 참여            │
│          │              │ (리빙랩 방식)            │
└──────────┴──────────────┴──────────────────────────┘
```

**효과**: 차별화 명확, 창의성 극대화
